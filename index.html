Web Publisher

HTML과 CSS를 다루며 또한 더 나아가 스크립트와 제이쿼리를 전문적으로 다루는 사람입니다.
웹표준과 웹접근성을 준수하며, 크로스브라우징을 통해 어떤 웹 브라우저를 사용하더라도 웹 페이지가 동일하게 보이도록 만들며, 사용에 문제가 생기지 않도록 하는 것이 중요합니다.
즉, 모든 사용자들을 생각하며 웹을 만드는 사람이 웹 퍼블리셔 입니다.

Even when dealing with HTML and CSS, it is the person who deal professionally with further scripts and J-query.
Compliance with Web standards and Web accessibility, even using any Web browser through a cross-browsing, to create as Web pages look the same, it is important that you do not cause a problem to use.
In other words, think of all the users, people who make the Web is a web publisher
